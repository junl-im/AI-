# v1.6.5 Patch Report

## 문제

기존 세로 렌더는 고정 위치 크롭만 제공해 인물이 좌우로 이동하면 피사체가 화면 밖으로 잘릴 수 있었고, 자막 위치와 피사체 위치도 서로 고려하지 않았습니다.

## 해결

- 영상 움직임 분석에 공간 중심·확산도·신뢰도 추가
- 모션 기반 추적선과 브라우저·MediaPipe 얼굴 감지 adapter 추가
- 얼굴 감지 실패 시 모션 중심 자동 대체
- 추적점 보간·평활화·최대 이동 제한
- 자막 위치를 피하는 9:16 동적 크롭
- 미리보기·재생·썸네일·최종 렌더 경로 통합
- 편집 단계 지연 적재와 원본 교체 시 추적 작업 취소

## 결과

- 별도 외부 서버 없이 기본 모션 스마트 리프레임 동작
- 선택형 로컬 얼굴 감지 연계 준비
- 초기 스크립트 예산과 기존 중앙 크롭 fallback 유지
- 실제 30분 1080p 영상의 변경 경로 집중 감사 통과
