- HTML, 자산 쿼리, 서비스워커 캐시와 package 버전이 v1.3.6으로 일치
- 서비스워커 등록·업데이트 확인이 단일 모듈에만 존재
- HTTPS·localhost 외의 비보안 HTTP에서는 등록을 건너뜀
- 등록 성공·실패·중복 방지·실패 후 재시도 확인

## 프로젝트·예외 입력 검사

- 프로젝트 스키마 v3 노출과 미래 스키마 거부
- 후보·자막 개수, 문자열, 프로젝트 파일과 자막 파일 크기 상한 확인
- 비정상 점수·시간 구간 보정과 최대 미디어 길이 상한 확인
- 알 수 없는 설정, 중첩 위험 키와 프로토타입 오염 키 제거
- 같은 파일 재선택 가능
- 지연 자동 분석이 이전 미디어 세션 결과를 덮어쓰지 않음
- 프로젝트에서 복구한 설정이 저장소에 반영됨
- 세션 복구가 동일한 프로젝트 검증을 재사용함

- 파일명·후보 제목·렌더 라벨은 HTML이 아닌 텍스트로 출력됨
- 외부 후보 ID를 CSS 선택자로 조립하지 않고 `dataset.id` 정확 비교로 연결함

- 20초 MP3: 분석 → 추천 → 선택 → 약 2초 렌더 → 다운로드 완료
- 20초 MP4: 오디오·움직임 분석 → 추천 → 선택 → 약 2초 렌더 → 다운로드 완료
- 렌더 취소: cancelled 1, 다운로드 0, 활성 operation 0
