# AI 쇼츠 스튜디오 v1.3.0 QA REPORT

## 요약

v1.3.0은 자동 검사 **118/118**과 실제 Chromium PC·모바일 런타임 감사를 통과했습니다. 이번 검수는 현재 단계 네온 랜딩, 지속 강조, 비동기 분석·미리보기·렌더 작업 소유권, 취소 신호 전파와 기존 메뉴·레이아웃 회귀를 함께 확인했습니다.

## 주요 검증

### 현재 단계 네온 랜딩

- `active-stage-beacon.css`가 최종 시각 오버라이드로 로드됩니다.
- 현재 패널에 네온 레일과 진행 중 칩이 유지됩니다.
- 새 단계 진입 순간에만 랜딩 스윕과 글로우가 한 번 실행됩니다.
- 랜딩 종료 뒤 애니메이션 클래스는 제거되고 현재 단계 테두리는 유지됩니다.
- 모바일과 `prefers-reduced-motion` 폴백이 존재합니다.
- 단계 변경은 `aria-live=polite`로 안내됩니다.

### 작업 소유권·취소

- 분석·미리보기·렌더 채널은 각각 하나의 활성 작업만 가집니다.
- 새 원본 파일이 들어오면 이전 미디어 세션 작업이 취소됩니다.
- 오디오 워커와 영상 seek·샘플링에 AbortSignal이 전달됩니다.
- 렌더 큐와 MediaRecorder가 취소 신호를 받고 `cancelled` 상태로 종료됩니다.
- 오래된 비동기 결과는 상태 반영과 다운로드 직전에 거부됩니다.
- 성공·실패·취소 경로에서 RAF·interval·timeout·MediaStreamTrack과 볼륨을 정리합니다.
