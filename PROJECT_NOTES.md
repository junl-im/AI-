# PROJECT NOTES - v0.6.0

## 이번 판단

상용 프로그램처럼 보이려면 추천 알고리즘뿐 아니라 마지막 출력물의 화면 질감이 중요합니다. v0.6.0은 영상 결과물에 직접 반영되는 품질 컨트롤을 먼저 넣었습니다.

## 설계 원칙

- 무료 기능만 사용
- 원본 파일은 서버 업로드 없음
- 미리보기와 내보내기의 렌더링 경로를 최대한 공유
- 안전영역 가이드는 출력물이 아니라 제작 보조로 취급
- 품질 설정은 프로젝트 JSON에 포함

## 알려진 한계

- 브라우저별 MediaRecorder 코덱 차이로 확장자와 품질이 다를 수 있음
- captureStream 오디오 트랙이 볼륨 페이드를 반영하지 않는 브라우저가 있을 수 있음
- 고해상도 긴 영상은 모바일에서 메모리 부담이 큼
