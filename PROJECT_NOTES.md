# PROJECT NOTES - AI 쇼츠 제작 스튜디오 v1.1.0

## 개발 방향

상용 프로그램처럼 보이려면 후보 선택 단계가 명확해야 합니다. v1.1.0은 후보 탭에 비교 보드를 추가해 사용자가 추천 결과를 더 빠르게 이해하고 선택할 수 있게 했습니다.

## 설계 메모

- `candidate-preview-pro.js`는 기존 추천 엔진을 바꾸지 않습니다.
- 비교 카드 클릭은 기존 `.recommendation-card` 클릭을 위임 호출합니다.
- 선택 후보 HUD는 미리보기 탭에만 표시되는 보조 안내입니다.
- 정렬 버튼은 Candidate Pro Board의 상위 비교 순서를 바꿉니다.
- 원본 추천 카드 목록은 그대로 유지해 회귀 위험을 줄였습니다.

## 유지해야 할 원칙

```text
├─ 상단은 소개/브랜드만
├─ Dock은 이동 메뉴만
├─ 후보 선택은 후보 탭 중심
├─ 선택 후 미리보기 자동 연결
├─ 화면 이동은 motion-stability.js 중심
└─ 저장은 렌더 큐와 품질 프리셋 중심
```
