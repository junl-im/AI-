# PROJECT NOTES - v0.7.0

## 이번 판단

상용 프로그램처럼 보이려면 추천 알고리즘뿐 아니라 마지막 출력물의 화면 질감이 중요합니다. v0.7.0은 영상 결과물에 직접 반영되는 품질 컨트롤을 먼저 넣었습니다.

## 설계 원칙

- 무료 기능만 사용
- 원본 파일은 서버 업로드 없음
- 미리보기와 내보내기의 렌더링 경로를 최대한 공유
- 안전영역 가이드는 출력물이 아니라 제작 보조로 취급
- 품질 설정은 프로젝트 JSON에 포함

## 알려진 한계

- 브라우저별 MediaRecorder 코덱 차이로 확장자와 품질이 다를 수 있음
- captureStream 오디오 트랙이 볼륨 페이드를 반영하지 않는 브라우저가 있을 수 있음
- 고해상도 긴 영상은 모바일에서 메모리 부담이 큼


## v0.7.0 자동 컷 설계

컷 후보는 세 종류입니다. `beat`는 오디오 transientNorm 피크, `motion`은 영상 diffNorm 피크, `silence-exit`은 무음 구간 종료 지점입니다. UI는 컷 후보 상위 10개를 칩으로 보여주며, 추천 후보에는 컷 밀도 기반 템포 점수와 무음 겹침 위험을 반영합니다.

다음 단계에서는 파형 오버레이에 컷 마커를 직접 그리거나, 컷 후보를 클릭하면 재생 위치가 이동하는 기능을 붙이면 좋습니다.


## v0.8.0 개발 노트

컷 포인트가 패널 칩에만 있을 때는 사용자가 실제 파형과의 관계를 파악하기 어렵다. 이번 버전은 컷 후보를 파형 위에 올려 추천 구간, 무음 영역, 비트/장면 전환 마커를 한 화면에서 볼 수 있도록 했다.

향후 확장 후보:

- 마커 타입별 표시/숨김 토글
- 컷 마커 드래그로 수동 컷 포인트 생성
- 선택 구간 내부 컷 밀도 기반 자동 분할
- 여러 구간을 하나의 릴스/쇼츠 시퀀스로 조합
