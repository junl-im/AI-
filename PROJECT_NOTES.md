# Project Notes v0.9.3

## 방향

상용 프로그램처럼 보이려면 UI 장식보다 흐름 안정성과 내부 구조가 먼저입니다. v0.9.3은 기능 확장 속도를 잠시 낮추고 엔진 계약, 캐시, 안정성 감사, 추천 신뢰도 계산을 추가했습니다.

## 다음 개발 후보

- 엔진 설정 탭: 안정 우선/품질 우선/최대 성능 모드 선택
- 추천 비교 모드: 후보 2~3개를 나란히 비교
- 렌더 큐: 여러 후보 저장 시 진행률/취소/재시도
- 자막 엔진 모듈 분리: 수동 자막, SRT, 향후 Whisper 모듈 교체 가능 구조

## 설계 원칙

- 기능은 모듈로 추가한다.
- 상단은 사용자 설명, 상세 패치 정보는 문서에 둔다.
- Dock은 탭 이동만 담당하고, 각 작업 액션은 탭 안에서 처리한다.
- 원본 미디어는 서버로 업로드하지 않는다.
