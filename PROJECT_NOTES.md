# PROJECT_NOTES v0.9.5

## UX 원칙

- 상단은 프로그램 소개와 브랜드만 보여준다.
- 추천 생성은 한 곳에서만 실행한다.
- 후보 선택은 사용자가 명확히 알 수 있어야 한다.
- 선택 후 다음 단계는 자동 연결한다.
- 긴 화면 나열 대신 하단 8탭 Dock으로 구간을 전환한다.

## HyperConnect flow

```text
File → Auto Analysis → Recommend → Select → Preview → Refine → Export
```

## 유지 정책

- 무료 로컬 처리.
- 모듈형 엔진 구조 유지.
- 화면 내 패치 노트 노출 금지.
- 패치 내용은 CHANGELOG, HANDOFF, PROJECT_NOTES로 분리.
