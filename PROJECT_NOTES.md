# Project Notes - v1.0.0

이번 패치의 핵심은 “AI 쇼츠 제작 스튜디오의 첫인상”입니다.

## 디자인 방향

- 기존 샘플의 움직이는 배경/물결감은 참고한다.
- 단, 이 앱은 마스터링 프로그램이 아니므로 오디오 미터나 물결을 전면에 두지 않는다.
- 쇼츠 제작 앱답게 필름 스트립, 카메라 프레임, 영상 슬라이드 느낌을 사용한다.
- 배경 톤은 기존의 어두운 글래스/네온 분위기를 유지한다.

## 고정된 흐름

1. 파일 열기
2. 자동 분석
3. 추천 탭에서 추천 생성
4. 후보 탭에서 구간 선택
5. 미리보기 탭으로 자동 연결
6. 파형/컷/편집/저장으로 이동

## 주의

- 상단은 제품 소개와 브랜드 첫인상만 담당한다.
- 패치 정보, 내부 엔진명, 개발자용 설명은 상단에 넣지 않는다.
- 애니메이션은 CSS만 사용하고 외부 리소스를 추가하지 않는다.
- `prefers-reduced-motion` 대응을 유지한다.
