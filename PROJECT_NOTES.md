# PROJECT NOTES - AI 쇼츠 제작 스튜디오 v1.0.8

## 방향

이번 버전은 상용 프로그램처럼 “저장 직전 확신”을 주는 쪽에 초점을 맞췄다.

## UX 원칙

- 후보를 선택하면 미리보기와 저장 상태가 자동으로 따라온다.
- 저장 버튼을 누르기 전에 현재 구간과 예상 정보를 한 번 더 보여준다.
- 화면 이동은 `motion-stability.js` 중심 구조를 유지한다.
- 새 보조 UI는 기존 탭/패널 구조를 덮어쓰지 않고 런타임으로 부착한다.

## 새 모듈

```text
save-readiness.js
```

역할:

- 선택 후보 확인
- 미리보기 준비 스트립 생성
- 저장 준비 패널 생성
- 길이/자막/예상 용량 표시
- 기존 HyperFlow 탭 API와 연결

## 다음 후보

v1.0.9에서는 저장 품질 프리셋, 렌더 예상 시간, 모바일 저장 실패 가이드를 추가하면 좋다.
