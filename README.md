# AI 쇼츠 제작 스튜디오 v1.0.5

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠 후보 추천·후보 선택·세로 미리보기·편집·저장까지 이어주는 로컬 웹 스튜디오입니다.

## 이번 업데이트

v1.0.5는 글라스 UI 적용 이후 실제 사용감을 다시 점검한 **Workspace Comfort 패치**입니다.

- Dock 탭을 눌렀을 때 해당 작업 패널이 화면 위쪽으로 더 확실하게 올라오도록 보강했습니다.
- 현재 열린 작업 패널에 짧은 하이라이트를 넣어 “어디가 열린 화면인지” 바로 보이게 했습니다.
- 후보 카드가 단순 정보 카드가 아니라 “선택 가능한 카드”처럼 보이도록 CTA와 선택 배지를 정리했습니다.
- 추천 안내 문구가 여러 flow 모듈에 의해 흔들리지 않도록 안정화했습니다.
- PC Dock은 8개 탭 한 줄 구조를 유지하면서 글자 가독성과 버튼 최소 폭을 다시 보강했습니다.
- 글라스 UI 표면은 유지하되, 후보/패널/Dock의 대비를 높여 편집 중 가독성을 개선했습니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 검수

```bash
npm run check
```

현재 검수: 75/75 통과.

## 작업 흐름

```text
파일 열기
→ 자동 분석
→ 추천 생성
→ 후보 선택
→ 미리보기
→ 파형 / 컷 / 편집 / 저장
```

Dock은 단순 연결 메뉴가 아니라, 사용자가 누른 작업 패널을 화면 전면으로 부드럽게 끌어올리는 하단 작업 내비게이션입니다.

## 모듈형 엔진

분석, 추천, 컷, 렌더, 안정성 점검은 `src/engine/`, `src/render/`, `src/ui/` 아래 모듈로 분리합니다. 새 기능은 앱 본문에 직접 쌓지 말고 모듈과 QA를 함께 추가합니다.
