# AI 쇼츠 제작 스튜디오 v1.1.3

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠 후보 추천·세로 미리보기·자막·저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.1.3 핵심

- 저장 완료 센터
- 렌더 완료/실패/부분 완료 상태 표시
- 실패 재시도와 진단 복사 바로가기
- 완료 후 미리보기/후보/저장 흐름 연결
- 기존 글라스 UI, 하단 Dock, 후보 핀, 세션 복구 유지

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 검수

```bash
npm run check
```

## 메모

원본 영상/음원은 브라우저 보안상 세션 저장에 포함되지 않습니다. 프로젝트나 세션 복구를 이어가려면 같은 원본 파일을 다시 열어주세요.

## 모듈형 엔진

분석, 추천, 자동 컷, 렌더 품질, 저장 큐, 세션 복구, 완료 센터는 각각 독립 모듈로 분리되어 있습니다. 새 기능은 가능한 한 기존 흐름을 직접 덮어쓰지 않고 모듈을 추가해 연결하는 방식으로 확장합니다.
