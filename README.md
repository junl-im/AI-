# AI 쇼츠 제작 스튜디오 v1.3.1

음악이나 영상을 브라우저 안에서 분석하고, 하이라이트 추천부터 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.3.1 핵심 패치

이번 버전은 PC의 3열 작업실을 사용자가 직접 조절할 수 있게 만들고, 작업 목적에 따라 화면을 넓게 전환하는 집중 보기를 추가했습니다.

- 3열 사이의 세로 핸들을 드래그해 왼쪽·가운데·오른쪽 폭을 조절합니다.
- 리사이저를 선택한 뒤 방향키로 미세 조절하고, Shift+방향키로 크게 조절할 수 있습니다.
- Home 키, 더블 클릭 또는 `초기화` 버튼으로 기본 비율을 복원합니다.
- 조절한 열 비율은 브라우저에 저장됩니다.
- `미리보기 집중`은 세로 미리보기를 크게 유지하고 후보·편집·저장을 오른쪽에 연결합니다.
- `파형 확대`는 파형과 컷 편집을 전체 폭 중심으로 재배치합니다.
- 집중 모드는 일시적인 작업 보기이며 다음 실행에서는 항상 3열 균형으로 시작합니다.
- 모바일과 1180px 미만 화면에서는 조절 UI가 노출되지 않습니다.
- 기존 메뉴 이동, 단계 네온 랜딩, 비동기 작업 취소와 엔진 안전성은 그대로 유지됩니다.

## 배포 파일

```bash
npm run package
```

- `dist/ai-shorts-studio-v1.3.1-release.zip`: 전체 설치용
- `dist/ai-shorts-studio-v1.3.1-patch-from-v1.3.0.zip`: v1.3.0 덮어쓰기용

패치 ZIP은 기존 v1.3.0 폴더 최상위에 풀고 파일 교체를 허용합니다. 다른 버전에는 적용하지 않습니다.

## PC 작업실 조작

- **드래그:** 두 세로 리사이저를 좌우로 이동
- **키보드:** 좌우 방향키, Shift+방향키
- **초기화:** Home, 리사이저 더블 클릭, 상단 `초기화`
- **집중 보기:** `미리보기 집중`, `파형 확대`

집중 보기에서 호환되지 않는 메뉴로 이동하면 자동으로 3열 균형 배치로 복귀합니다.

## 알려진 제한

실제 대용량 MP4 장시간 분석·렌더·저장, 모바일 Safari와 인앱 브라우저의 MediaRecorder 출력 형식은 실기기에서 추가 검증해야 합니다. 관리형 Chromium의 로컬 주소 제한 때문에 런타임 감사에서는 동일한 HTML·CSS·스크립트를 감사 문서에 직접 주입해 실행했습니다.
