# AI 쇼츠 제작 스튜디오 v1.2.2

음악이나 영상을 브라우저 안에서 분석하고, 쇼츠 후보 추천부터 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.2.2 핵심 패치

- 상단을 장식이 많은 배너에서 **에디토리얼 마스트헤드**로 다시 설계했습니다.
- 버전과 `Design by 곰같은여우`는 둥근 배지가 아니라 같은 기준선 위의 텍스트 메타 정보로 표시합니다.
- 필름 스트립, 뷰파인더, 회전 릴과 중첩 테두리를 제거하고 타이포그래피·여백·한 개의 구분선으로 깊이를 만듭니다.
- PC의 오른쪽 시작 영역은 별도 카드가 아니라 작업실에 연결된 시작 레일이며, 기존 `fileInput`을 그대로 사용합니다.
- 모바일은 `BUILD v1.2.2`와 `DESIGNED BY 곰같은여우`, 제품 설명과 4단계 진행 안내만 간결하게 보여줍니다.
- 상단 CSS를 두 파일에서 한 파일로 통합해 초기 CSS 요청을 39개에서 38개로 줄였습니다.
- 첫 화면 밖의 자막·품질·프로젝트 영역은 브라우저가 지원할 때 페인트를 지연해 초기 렌더 부담을 낮춥니다.
- v1.2.0에서 해결한 Observer 피드백 루프와 페이지 응답성 가드는 그대로 유지합니다.

## 화면별 사용 흐름

### PC

상단의 `새 프로젝트 시작` 또는 작업실의 `파일 열기`로 원본을 선택합니다. 첫 화면의 3열 Prime 작업실에서 추천 설정, 후보 목록, 세로 미리보기와 파형을 함께 확인하며, 하단 메뉴바는 해당 작업 카드로 빠르게 이동합니다.

### 모바일

상단에서 제품의 성격과 제작 순서를 확인한 뒤 하단 메뉴바의 `📂 파일 열기`를 눌러 시작합니다. 본문에는 중복 파일 불러오기 카드를 두지 않습니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다. 파일 탐색기에서 `index.html`을 직접 여는 것보다 로컬 서버 실행을 권장합니다.

## 검수

```bash
npm run check
```

v1.2.2 기준 문법, DOM 앵커, 렌더·추천·저장 모듈, PC/모바일 레이아웃, 응답성 회귀 방지, 에디토리얼 상단과 뷰포트 렌더 최적화까지 105개 검사를 실행합니다.

## 업데이트 확인

상단 버전 텍스트를 누르면 현재 버전, 빌드 키, 서비스워커, 캐시와 엔진 프로필을 확인할 수 있습니다. 이전 캐시가 남아 있으면 정보 창에서 이전 캐시 정리를 실행한 뒤 새로고침합니다.

## 모듈형 엔진

분석, 추천, 컷 감지, 렌더 품질, 캐시와 안정성 감사 모듈의 계약은 유지합니다. 이번 패치는 상단 HTML·CSS, 첫 페인트 최적화와 버전 동기화 중심이며 분석·렌더 엔진 로직은 변경하지 않았습니다.
