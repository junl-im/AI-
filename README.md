# AI 쇼츠 제작 스튜디오 v1.2.4

음악이나 영상을 브라우저 안에서 분석하고, 하이라이트 추천부터 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.2.4 핵심 패치

- PC에서 작업이 진행될 때 메뉴바와 해당 작업 패널이 함께 따라가는 내비게이션 포커스를 추가했습니다.
- 오래된 메뉴 모듈이 새 진행 상태를 다시 `파일`로 덮어쓰던 실제 런타임 충돌을 수정했습니다.
- 파일 선택 → 자동 분석 화면 → 추천 완료 후 후보 → 후보 선택 후 미리보기 → 렌더 시작 후 저장 화면으로 연결됩니다.
- 현재 대상 패널은 PC Prime 작업실에서 외곽선과 `현재 작업` 레이블로 분명하게 표시됩니다.
- 상단 버전 옆에 `모바일 · PC 호환`을 표시하며 모바일에서도 유지합니다.
- 메뉴 아이콘을 운영체제별 컬러 이모지 대신 `＋ ✦ ◆ ▶ ∿ ✂ ◫ ↓`의 단색 스튜디오 글리프로 통일했습니다. 파형·저장·업데이트 보조 문구도 같은 체계로 맞췄습니다.
- 모바일 업데이트 알림은 상단 메타를 가리지 않도록 하단 메뉴바 위에 표시합니다.
- v1.2.3의 Shorts Pulse 상단, 3열 Prime 작업실, 모바일 진행 중심 화면과 응답성 가드는 유지합니다.

## 화면별 사용 흐름

### PC

상단의 `새 쇼츠 만들기` 또는 작업실의 `파일 열기`로 원본을 선택합니다. 진행 단계가 바뀌면 메뉴바 활성 상태와 작업 카드 강조가 함께 이동합니다. 메뉴를 직접 눌러도 해당 카드가 화면으로 이동하고 강조됩니다.

### 모바일

상단에서 버전과 모바일·PC 호환 정보를 확인한 뒤 하단 메뉴바의 `파일 열기`를 눌러 시작합니다. PC용 세로 프레임 액션은 숨기고 진행 순서와 현재 작업 화면에 집중합니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다. 파일 탐색기에서 `index.html`을 직접 여는 것보다 로컬 서버 실행을 권장합니다.

## 검수

```bash
npm run check
```

v1.2.4 기준 문법, DOM 앵커, 분석·추천·렌더·저장 모듈, PC/모바일 레이아웃, 진행 내비게이션, 메뉴 아이콘 일관성과 응답성 회귀 검사를 실행합니다. 별도 Chromium 감사에서는 합성 20초 WAV로 파일 → 추천 → 후보 → 미리보기 연결을 확인했습니다.

## 업데이트 확인

상단 버전 텍스트를 누르면 현재 버전, 빌드 키, 서비스워커, 캐시와 엔진 프로필을 확인할 수 있습니다. 이전 캐시가 남아 있으면 정보 창에서 이전 캐시 정리를 실행한 뒤 새로고침합니다.

## 모듈형 엔진

분석, 추천, 컷 감지, 렌더 품질, 캐시와 안정성 감사 모듈의 계약은 유지합니다. 이번 패치는 진행 내비게이션, 상단 호환 정보와 아이콘 체계 중심이며 분석·렌더 알고리즘은 변경하지 않았습니다.
