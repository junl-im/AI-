# AI 쇼츠 제작 스튜디오 v1.1.5

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠로 쓰기 좋은 후보 구간을 추천해 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## 이번 버전 핵심

- 상단은 좌측 `v1.1.5`, 가운데 `PC · 모바일 호환`, 우측 `Design by 곰같은여우` 한 줄 구조로 보여줍니다.
- YouTube / Reels / TikTok 감성의 셔터 플래시 글라스 연출을 추가했습니다.
- 본 화면은 단순 사용 방법 중심으로 정리했습니다.
- 실제 작업은 하단 Dock을 눌러 각 작업 패널로 이동합니다.
- Dock 이동 시 해당 작업 패널이 전면에 오도록 단일 Flow Director가 담당합니다.
- 특정 구간에서 보이던 화면 떨림을 줄이기 위해 `flow-command-bridge.js`로 Dock 클릭 이벤트를 먼저 잡고, 레거시 탭 핸들러의 중복 스크롤을 차단했습니다.

## 사용 흐름

1. 하단 Dock의 `📂 파일 열기`로 원본을 선택합니다.
2. 파일을 열면 자동 분석이 시작됩니다.
3. `✨ 추천`에서 후보를 생성합니다.
4. `🎯 후보`에서 마음에 드는 구간을 선택합니다.
5. `📱 미리보기`, `〰️ 파형`, `✂️ 컷`, `🎛️ 편집`, `📦 저장`으로 이어갑니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 모듈형 엔진

분석, 추천, 컷 감지, 렌더 품질, 세션 복구, 저장 큐를 독립 모듈로 분리해 유지보수와 확장을 쉽게 만들었습니다. v1.1.5에서는 화면 이동과 패널 표시를 `Final Flow Director`가 최종 관리해 UI 흔들림을 줄입니다.
