# AI 쇼츠 제작 스튜디오 v1.0.3

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠 후보 추천·후보 선택·세로 미리보기·편집·저장까지 이어주는 로컬 웹 스튜디오입니다.

## 이번 업데이트

- PC 하단 Dock이 좁아져 글자가 보이지 않던 문제를 수정했습니다.
- PC에서는 8개 탭이 한 줄에서 읽히도록 Dock 폭과 버튼 크기를 다시 잡았습니다.
- 태블릿/모바일은 4개씩 두 줄 구조를 유지합니다.
- Dock에서 추천, 후보, 미리보기, 파형, 컷, 편집, 저장을 누르면 해당 작업 화면이 실제로 화면 상단 가까이 올라오도록 강제 reveal을 적용했습니다.
- 후보 안내 문구가 서로 다른 모듈에서 번갈아 바뀌며 반짝이는 문제를 줄이기 위해 안내 문구를 통일했습니다.
- 구간 선택, 미리보기, 내보내기 버튼은 다시 작은 작업 버튼 크기로 고정했습니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 검수

```bash
npm run check
```

현재 검수: 72/72 통과.

## 작업 흐름

```text
파일 열기
→ 자동 분석
→ 추천 생성
→ 후보 선택
→ 미리보기
→ 파형 / 컷 / 편집 / 저장
```

Dock은 단순 연결 상태만 바꾸는 메뉴가 아니라, 사용자가 누른 작업 화면으로 자연스럽게 이동시키는 하단 작업 내비게이션입니다.

## 모듈형 엔진

분석, 추천, 컷, 렌더, 안정성 점검은 `src/engine/`, `src/render/`, `src/ui/` 아래 모듈로 분리합니다. 새 기능은 앱 본문에 직접 쌓지 말고 모듈과 QA를 함께 추가합니다.
