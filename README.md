# AI 쇼츠 제작 스튜디오 v1.0.8

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠로 쓰기 좋은 후보 구간을 추천해 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.0.8 핵심

```text
파일 열기
→ 자동 분석
→ 추천 생성
→ 후보 선택
→ 미리보기 확인
→ 저장 전 체크리스트
→ 쇼츠 저장
```

## 이번 버전 추가

- 미리보기 탭에 선택 후보 준비 스트립 추가.
- 저장 탭에 저장 전 체크리스트 추가.
- 선택 구간, 길이, 자막 상태, 예상 용량 표시.
- 후보/미리보기/편집으로 바로 이동하는 보조 버튼 추가.
- 화면 떨림 방지 구조 유지.

## 실행

```bash
npm run serve
```

브라우저에서:

```text
http://localhost:8080
```

## 검수

```bash
npm run check
```

결과:

```text

```

## 메모

이 프로젝트는 무료 로컬 브라우저 기능을 중심으로 동작합니다. 파일은 사용자의 브라우저 안에서 처리되며, 별도 서버 업로드를 전제로 하지 않습니다.

## 모듈형 엔진

분석, 추천, 컷 감지, 렌더링, 저장 준비 안내는 각각 독립 모듈로 분리되어 있습니다. 이 구조는 이후 무료 로컬 엔진을 더 강화하거나, 필요 시 별도 고급 분석 모듈을 추가하기 쉽게 하기 위한 기반입니다.
