# AI 쇼츠 제작 스튜디오 v0.9.7

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠 후보 추천부터 구간 선택, 세로 미리보기, 자막/편집, 저장까지 이어주는 로컬 웹 스튜디오입니다.

## 핵심 흐름

```text
📂 파일 열기
→ 자동 분석
→ ✨ 추천 생성
→ 👆 후보 탭에서 구간 선택
→ 📱 미리보기 자동 연결
→ 〰️ 파형 / ✂️ 컷 / 🎛️ 편집 / ⬇️ 저장
```

## v0.9.7 안정화

- 후보 선택 전용 탭 추가
- 탭 클릭 시 최상단으로 튀는 문제 수정
- 추천 생성 후 후보 탭 자동 이동
- 후보 선택 후 미리보기 탭 자동 이동
- 구간 선택/미리보기/저장 버튼 크기 축소
- 하단 Dock 4+4 구조 유지
- 상단은 프로그램 소개와 버전/디자인 표기만 유지

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 검수

```bash
npm run check
```

## 모듈형 엔진

분석, 추천, 컷, 렌더, 안정성 점검은 독립 모듈로 분리되어 있습니다. 기능을 추가할 때도 기존 흐름을 깨지 않도록 모듈 단위로 확장합니다.
