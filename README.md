# AI 쇼츠 제작 스튜디오 v1.6.4

브라우저에서 미디어 분석, 쇼츠 후보 추천, 자막 편집, 세로 영상 렌더링과 선택형 Local AI를 수행하는 로컬 우선 제작 도구입니다. v1.6.4는 저장소 문제 안내부터 해결 완료 후 작업실 복귀까지의 흐름을 완결하고, 삭제 전에 실제 영향 범위를 보여줍니다.

## 핵심 기능

- 모듈형 엔진 기반 오디오·영상 분석, 추천, 캐시, 안정성 감사
- 자동 컷, 후보 고정, 자막·SRT 편집, 9:16 렌더 큐
- 옵션 signature 기반 분석 캐시와 namespace 선택 정리
- 세션 순환 백업과 중요 백업 내보내기·가져오기
- 서비스워커 앱 셸 SHA-256 무결성 검사와 복구
- Ollama·llama.cpp 구조화 카피 생성
- whisper.cpp·OpenAI 호환 로컬 전사 및 SRT 생성
- Local AI 모델 digest 고정, 직렬 작업 큐, 취소·시간 제한·진행률

## v1.6.4 저장소 복구 UX

- 저장 공간 또는 오프라인 문제가 생기면 최하단 상태 영역으로 한 번 안내합니다.
- `문제 자동 해결`이 성공하고 상태가 정상으로 바뀌면 페이지 최상단으로 돌아갑니다.
- 취소하거나 문제가 남으면 하단에서 계속 조치할 수 있습니다.
- 저장소 정리 전 오래된 백업 수, 예상 로컬 용량, 이전 오프라인 캐시 수를 표시합니다.
- 현재 프로젝트, 원본 파일, 최신 복구 데이터, 현재 오프라인 셸은 유지합니다.

## 로컬 AI 원칙

1. 페이지를 열기만 해서는 AI 서버에 요청하지 않습니다.
2. 연결 확인·카피 생성·전사는 사용자의 명시적 동작 뒤에만 실행됩니다.
3. 기본 설정에서는 loopback 주소만 허용합니다.
4. 외부 CDN, 원격 모델, API 키는 배포본에 포함하지 않습니다.
5. Local AI가 없어도 기존 분석·편집·렌더 기능은 계속 동작합니다.
6. Local AI 모듈은 해당 영역을 처음 사용할 때만 단계 적재합니다.

## 실행과 검수

```bash
npm run serve
npm test
```

브라우저에서 `http://127.0.0.1:8080`을 엽니다. 등록 자동 검사 224개와 저장소 복구 전용 데스크톱·모바일 브라우저 감사를 포함합니다.

## 패키징

```bash
npm run package:full
PATCH_BASE_REF=e51cbce PATCH_FROM_VERSION=1.6.3 npm run package:patch
```

전체 ZIP은 실행·문서·QA 파일을 포함하고 `.git`, `node_modules`, 중첩 `dist`를 제외합니다. 패치 ZIP은 v1.6.3 기준 변경·신규 파일만 포함합니다.

## 알려진 제한

- AI 모델과 로컬 서버 실행 파일은 배포본에 포함되지 않습니다.
- 실기기 Safari·Samsung Internet, 물리 GPU·NPU는 별도 검증이 필요합니다.
- 브라우저 보안 정책 때문에 Local AI 서버의 CORS 설정이 필요할 수 있습니다.
- Cache Storage의 정확한 정리 예상 바이트는 브라우저 API 제한으로 표시하지 않습니다.
- 프로세스 메모리 수치는 headless Chromium 환경의 보조 지표입니다.
