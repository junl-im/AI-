# AI 쇼츠 제작 스튜디오 v1.1.8

음악이나 영상을 브라우저 안에서 분석하고, 쇼츠로 쓰기 좋은 구간 추천부터 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.1.8 핵심 패치

- 첫 화면을 **파일 열기 + 프로젝트 불러오기 + 4단계 제작 흐름** 중심으로 다시 정리했습니다.
- 중복 안내 카드를 줄이고, 미디어를 연 뒤에는 시작 안내 영역이 자동으로 접혀 편집 공간을 넓게 사용합니다.
- 700ms마다 실행되던 UX 동기화 폴링을 이벤트 기반 `requestAnimationFrame` 갱신으로 교체했습니다.
- 500ms마다 파형 선택 DOM을 다시 만들던 작업을 제거하고, 선택 구간이 실제로 달라졌을 때만 렌더링합니다.
- 저사양 기기, 데이터 절약 모드, 모션 감소 환경에서는 블러와 장식 애니메이션을 자동으로 줄입니다.
- 데스크톱 Dock 높이와 패널 여백을 줄여 실제 편집 영역을 더 확보했습니다.
- 기존 Update Sentinel, Engine Boost Profile, 8단계 Dock 흐름은 유지합니다.

## 사용 흐름

1. 첫 화면 또는 하단 Dock에서 `📂 파일 열기`를 선택합니다.
2. 파일을 열면 브라우저 안에서 자동 분석이 시작됩니다.
3. `✨ 추천`에서 후보를 생성합니다.
4. `🎯 후보`에서 마음에 드는 구간을 선택합니다.
5. `📱 미리보기`, `〰️ 파형`, `✂️ 컷`, `🎛️ 편집`, `📦 저장`으로 이어갑니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 검수

```bash
npm run check
```

v1.1.8 기준 문법, DOM 앵커, 렌더/추천/저장 모듈, 반응형 레이아웃, 성능 가드까지 101개 검사를 실행합니다.

## 업데이트 확인

상단 버전 배지를 누르면 프로그램 정보 창에서 현재 버전, 빌드 키, 서비스워커, 캐시, 엔진 프로필을 확인할 수 있습니다. 이전 캐시 정리와 진단 복사 기능도 기존과 동일하게 제공됩니다.

## 모듈형 엔진

분석, 추천, 컷 감지, 렌더 품질, 캐시, 안정성 감사를 독립 모듈로 유지합니다. v1.1.8은 엔진 계약을 바꾸지 않고 UI 동기화와 파형 오버레이의 런타임 비용을 줄였습니다.
