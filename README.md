# AI 쇼츠 제작 스튜디오 v1.0.2

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠 후보 추천·후보 선택·세로 미리보기·편집·저장까지 이어주는 로컬 웹 스튜디오입니다.

## 이번 업데이트

- 초연결 흐름을 다시 점검하고 `Flow Quality Gate` 런타임 가드를 추가했습니다.
- 활성 탭이 잘못 잡히거나 패널이 여러 개 펼쳐지는 상황을 자동 복구합니다.
- 현재 상태에 맞지 않는 탭으로 이동하면 파일 열기/추천/후보/미리보기 중 올바른 단계로 되돌립니다.
- 추천 생성 후 후보 탭, 후보 선택 후 미리보기 탭 연결을 다시 보강했습니다.
- 중복 액션 Dock과 레거시 모바일 액션바는 런타임에서도 숨기도록 추가 방어했습니다.
- JavaScript 런타임 오류와 Promise 오류를 진단 로그에 남기고 토스트로 알려줍니다.
- PC/모바일 반응형 레이아웃과 v1.0.1 시네마틱 히어로 디자인은 유지했습니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 검수

```bash
npm run check
```

현재 검수: 71/71 통과.

## 모듈형 엔진 원칙

분석, 추천, 컷, 렌더, 안정성 점검은 `src/engine/`, `src/render/`, `src/ui/` 아래의 모듈로 분리합니다. 앱 본문에 새 기능을 직접 쌓기보다 모듈을 추가하고 QA를 함께 붙이는 방향을 유지합니다.
