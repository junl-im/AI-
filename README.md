# AI 쇼츠 제작 스튜디오 v1.6.3

브라우저에서 미디어 분석, 쇼츠 후보 추천, 자막 편집, 세로 영상 렌더링과 선택형 Local AI를 수행하는 로컬 우선 제작 도구입니다. v1.6.3은 데스크톱 작업실을 현재 단계 중심으로 정리해 필요한 패널만 보여 주고, 언제든 전체 보기로 되돌릴 수 있게 합니다.

## 핵심 기능

- 오디오·영상 움직임 분석과 쇼츠 후보 추천
- 자동 컷, 후보 고정, 자막·SRT 편집, 9:16 렌더 큐
- 옵션 signature 기반 분석 캐시와 namespace 선택 정리
- 세션 순환 백업과 중요 백업 내보내기·가져오기
- 서비스워커 앱 셸 SHA-256 무결성 검사와 복구
- Ollama·llama.cpp 구조화 카피 생성
- whisper.cpp·OpenAI 호환 로컬 전사 및 SRT 생성
- Local AI 모델 digest 고정, 직렬 작업 큐, 취소·시간 제한·진행률

## v1.6.3 단계 집중 작업실

- 데스크톱 균형 보기에서 현재 작업을 주 패널로 확장합니다.
- 바로 이어지는 단계만 86px 지원 카드로 남기고, 나머지 패널은 작업 흐름에서 제외합니다.
- 지원 카드의 `이 단계 열기`로 해당 작업을 즉시 주 패널로 승격합니다.
- `전체 보기`로 기존 8개 작업 패널과 프로젝트·카피·Local AI 도구를 복원합니다.
- 미리보기·파형 전용 보기에서는 자동 단계 집중을 잠시 중지합니다.
- 모바일은 기존 단일 단계 흐름을 유지합니다.
- 하단 메뉴 실측 높이를 콘텐츠 여백에 반영해 겹침을 방지합니다.

## 로컬 AI 원칙

1. 페이지를 열기만 해서는 AI 서버에 요청하지 않습니다.
2. 연결 확인·카피 생성·전사는 사용자의 명시적 동작 뒤에만 실행됩니다.
3. 기본 설정에서는 loopback 주소만 허용합니다.
4. 외부 CDN, 원격 모델, API 키는 배포본에 포함하지 않습니다.
5. Local AI가 없어도 기존 분석·편집·렌더 기능은 계속 동작합니다.
6. Local AI 모듈은 해당 영역을 처음 사용할 때만 단계 적재합니다.

## 실행과 검수

```bash
npm run serve
npm test
```

브라우저에서 `http://127.0.0.1:8080`을 엽니다. v1.6.3은 등록 자동 검사 224개, 4개 viewport, 단계 집중 전용 데스크톱·모바일 감사, CSS ownership, 서비스워커 lifecycle, GPU/media capability와 프로세스 메모리 보조 감사를 포함합니다.

## 패키징

```bash
npm run package:full
PATCH_BASE_REF=66e5b42 PATCH_FROM_VERSION=1.6.2 npm run package:patch
```

전체 ZIP은 실행·문서·QA 파일을 포함하고 `.git`, `node_modules`, 중첩 `dist`를 제외합니다. 패치 ZIP은 v1.6.2 기준 변경·신규 파일만 포함합니다.

## 알려진 제한

- AI 모델과 로컬 서버 실행 파일은 배포본에 포함되지 않습니다.
- 실기기 Safari·Samsung Internet, 물리 GPU·NPU는 별도 검증이 필요합니다.
- 브라우저 보안 정책 때문에 Local AI 서버의 CORS 설정이 필요할 수 있습니다.
- 단계 집중은 데스크톱 균형 보기에서만 적용되며, 사용자가 전체 보기로 해제할 수 있습니다.
- 프로세스 메모리 수치는 headless Chromium 환경의 보조 지표입니다.
