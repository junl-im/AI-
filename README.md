# AI 쇼츠 제작 스튜디오 v1.2.9

음악이나 영상을 브라우저 안에서 분석하고, 하이라이트 추천부터 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.2.9 핵심 패치

이번 버전은 디자인 변경보다 **PC 메뉴바 가시성, 런타임 오류 감지, 단계 로더 일관성, 렌더 타이머 정리**에 집중한 안정화 릴리스입니다.

- PC 메뉴바 외부 프레임과 내부 레일의 서로 다른 너비 규칙을 통합했습니다.
- 1180·1280·1366·1920px에서 메뉴 8개가 모두 화면 안에 표시됩니다.
- 1180px 미만에서는 안전한 4열×2행으로 전환합니다.
- 지연 로더의 `editor`/`edit` 키 불일치와 v1.2.6 고정 캐시 키를 제거했습니다.
- 지연 모듈 로딩에 10초 타임아웃과 오류 진단을 추가했습니다.
- 전역 오류와 처리되지 않은 Promise 거절을 런타임 건강 상태에 기록합니다.
- 미리보기 재생 거절 시 RAF·interval을 시작하지 않도록 수정했습니다.
- 렌더 성공·실패·중단 모든 경로에서 타이머와 캡처 트랙을 정리합니다.
- 세션 자동 저장은 숨김 탭에서 멈추고 30초 주기로 완화했습니다.
- 제거된 `syncTopLine()` 잔존 호출로 발생하던 실제 브라우저 `ReferenceError`를 제거했습니다.
- 실제 Chromium 감사 결과를 자동 QA가 직접 검증하도록 추가했습니다.

## 배포 파일

```bash
npm run package
```

- `dist/ai-shorts-studio-v1.2.9-release.zip`: 전체 설치용
- `dist/ai-shorts-studio-v1.2.9-patch-from-v1.2.8.zip`: v1.2.8 덮어쓰기용

패치 ZIP은 기존 v1.2.8 폴더 최상위에 풀고 파일 교체를 허용합니다. 다른 버전에 적용하지 않습니다.

## 화면별 사용 흐름

### PC

상단 왼쪽에서 빌드와 기기 호환 정보를 확인하고, 오른쪽에서 디자인 서명을 확인할 수 있습니다. 하단 메뉴바의 8개 메뉴는 1366px급 일반 노트북 화면에서도 잘리지 않으며, 진행 단계가 바뀌면 활성 메뉴·현재 작업 카드·화면 위치가 함께 이동합니다.

### 모바일

`BUILD v1.2.9`, `모바일 · PC 호환`, `DESIGNED BY 곰같은여우`를 유지합니다. 파일 선택은 하단 메뉴바의 `파일 열기`가 담당하고, 메뉴는 4열×2행으로 표시됩니다.

## 아이콘 자산 구조

- SVG 원본: `assets/icons/studio/`
- CSS 매핑: `assets/css/icon-system.css`
- 사용 방식: `data-icon="upload"` 또는 `<span class="studio-icon" data-icon="upload"></span>`

핵심 UI에는 운영체제별 이모지나 문자 기호 대신 24×24 전용 SVG 자산을 사용합니다.

## 모듈형 엔진

오디오 특징 추출, 영상 모션 분석, 자동 컷, 추천 점수, 분석 캐시, 안정성 감사와 렌더 품질은 독립 모듈 계약으로 유지합니다. v1.2.9는 엔진 결과 형식을 바꾸지 않고 재생 실패·타이머 정리·런타임 오류 관측 경로를 보강했습니다.

## 단계별 UI 로딩

| 단계 | 준비되는 기능 | 시점 |
|---|---|---|
| Core | 앱 상태, 분석·추천·렌더 핵심, 메뉴 내비게이션 | 첫 화면 |
| Shell | 흐름 보정, UX 동기화, 작업 공간 안정화, 세션 연속성 | 유휴 시간 또는 첫 접근 |
| Editing | 범위 조절, 후보 비교·고정, 저장 준비, 품질 계획 | 후보·미리보기·컷·편집 접근 |
| Export | 저장 완료 센터 | 내보내기 접근 |

단계 키는 실제 메뉴 키와 동일한 `edit`를 사용하며, 버전 쿼리는 런타임 설정의 빌드 키에서 생성합니다.
