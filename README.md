# AI 쇼츠 제작 스튜디오 v1.6.9

브라우저에서 미디어 분석, 쇼츠 후보 추천, 자막 편집, 스마트 세로 크롭, 렌더링과 선택형 Local AI를 수행하는 모듈형 엔진 기반의 로컬 우선 제작 도구입니다.

v1.6.9은 미리보기 화면에서 피사체 위치를 직접 드래그하고 휠·핀치·키보드로 확대와 위치를 조절하는 직접 크롭 편집을 추가합니다. 조작 결과는 현재 재생 위치의 키프레임으로 저장되며 미리보기·썸네일·최종 렌더에 동일하게 적용됩니다.

## 실행
정적 웹 서버에서 프로젝트 루트를 제공합니다.

```bash
npm run serve
```

## 검사

```bash
npm run check
```
