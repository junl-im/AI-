# AI 쇼츠 제작 스튜디오 v0.9.5

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠 후보 추천·세로 미리보기·자막·저장까지 이어주는 로컬 웹 스튜디오입니다.

## v0.9.5 핵심

- 파일 열기 후 자동 분석 흐름 유지
- 상단 중복 추천 생성 버튼 제거
- 추천 생성 버튼은 추천 탭 안에 하나만 유지
- 추천 후보 카드에 `선택해서 미리보기` CTA 추가
- 후보 선택 시 미리보기 탭 자동 전환 흐름 강화
- 상단 정보 정리: 왼쪽 버전, 오른쪽 `Design by 곰같은여우`
- 개발자용 패치 정보는 화면이 아니라 문서로 분리
- 하단 8탭 Dock 기반 초연결 작업 흐름 유지

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 기본 흐름

```text
파일 열기
→ 자동 분석
→ 추천 탭 자동 이동
→ ✨ 추천 생성
→ 후보 카드 선택
→ 📱 미리보기 자동 이동
→ 파형/컷/편집/자막/저장 탭으로 후속 작업
```

## 모듈형 엔진

분석, 추천, 컷, 렌더, 안정성 점검은 모듈형 엔진 구조로 분리되어 있습니다. 새 분석기나 추천 점수 로직을 추가할 때는 `src/engine/` 계층을 우선 사용합니다.
