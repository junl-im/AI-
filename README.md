# AI 쇼츠 제작 스튜디오 v1.3.0

음악이나 영상을 브라우저 안에서 분석하고, 하이라이트 추천부터 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.3.0 핵심 패치

이번 버전은 **현재 진행 구역을 한눈에 보여주는 네온 랜딩 UI**와 **비동기 엔진 작업 소유권·취소 안전성**을 함께 강화했습니다.

- 파일 열기 → 추천 → 후보 → 미리보기 → 저장 단계가 바뀔 때 대상 패널 외곽에 밝은 네온 라인이 나타납니다.
- 단계 진입 순간에는 짧은 빛의 스윕이 한 번 흐르고, 이후에는 현재 작업 구역을 알리는 테두리가 유지됩니다.
- 패널 우측 상단에 `추천 생성 진행 중`, `미리보기 진행 중`처럼 현재 단계가 표시됩니다.
- 단계별로 시안·바이올렛·핑크·그린 계열의 미세한 색조 차이를 사용합니다.
- `prefers-reduced-motion` 환경에서는 스윕 애니메이션을 제거하고 고정 강조만 유지합니다.
- 새 파일을 열면 이전 분석·미리보기·렌더 작업을 무효화해 늦게 끝난 결과가 새 프로젝트를 덮어쓰지 못하게 했습니다.
- 오디오 워커, 비디오 모션 샘플링, 미리보기, 렌더 큐와 MediaRecorder에 취소 신호를 전달합니다.
- 렌더 취소 상태를 `cancelled`로 구분하고 타이머·트랙·볼륨을 같은 정리 경로에서 종료합니다.
- 실제 Chromium에서 PC·모바일 네온 랜딩, 지속 강조, 오류·Promise 거절·반복 RAF를 검증합니다.

## 배포 파일

```bash
npm run package
```

- `dist/ai-shorts-studio-v1.3.0-release.zip`: 전체 설치용
- `dist/ai-shorts-studio-v1.3.0-patch-from-v1.2.9.zip`: v1.2.9 덮어쓰기용

패치 ZIP은 기존 v1.2.9 폴더 최상위에 풀고 파일 교체를 허용합니다. 다른 버전에 적용하지 않습니다.

## 화면별 진행 표시

### PC

여러 작업 패널을 유지하면서 현재 단계 패널만 밝은 네온 외곽선과 진행 칩으로 강조합니다. 메뉴 활성, 화면 이동, 패널 강조가 같은 단계 키를 사용합니다.

### 모바일

활성 작업 패널을 우선 표시하고 같은 네온 라인을 더 얇게 적용합니다. 하단 메뉴바는 4열×2행을 유지하며 현재 단계와 패널 강조가 함께 이동합니다.

## 모듈형 엔진

분석·추천·렌더 기능은 모듈 레지스트리와 엔진 커널을 통해 연결됩니다. 각 모듈은 공통 계약과 성능 예산을 따르며, 분석 캐시·안정성 감사·프로 엔진 튜너가 동일한 상태 정보를 공유합니다.

## 비동기 작업 안전성

`AIShortsOperationCoordinator`가 분석·미리보기·렌더 작업에 고유 작업 ID와 미디어 세션 ID를 부여합니다. 새 원본이 열리거나 같은 종류의 새 작업이 시작되면 이전 작업은 취소되고, 완료 시점에 현재 작업인지 다시 검사한 뒤에만 상태와 파일을 저장합니다.
