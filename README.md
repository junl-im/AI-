# AI 쇼츠 제작 스튜디오 v1.1.7

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠로 쓰기 좋은 후보 구간을 추천해 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## 이번 버전 핵심

- `Update Sentinel`을 추가해 정보 모달에서 현재 버전, 빌드 키, 서비스워커, 캐시 상태, 엔진 프로필을 바로 확인할 수 있습니다.
- 이전 서비스워커 캐시가 남아 버전이 안 바뀐 것처럼 보일 때 사용할 수 있는 `업데이트 확인`, `이전 캐시 정리`, `진단 복사` 흐름을 추가했습니다.
- `Engine Boost Profile`을 추가해 브라우저 성능을 감지하고 `MAX-STABLE / PRO-STABLE / SAFE-STABLE` 프로필을 산정합니다.
- 상단 한 줄 구조는 좌측 `v1.1.7`, 가운데 `PC · 모바일 호환`, 우측 `Design by 곰같은여우`를 유지합니다.
- YouTube / Reels / TikTok 감성의 셔터 플래시 글라스 연출을 유지하되 작업 패널 애니메이션과 분리했습니다.
- Dock 이모지는 `📂 파일 열기 / ✨ 추천 / 🎯 후보 / 📱 미리보기 / 〰️ 파형 / ✂️ 컷 / 🎛️ 편집 / 📦 저장`으로 재정리했습니다.
- 본 화면은 단순 사용 방법 중심으로 유지하고, 실제 작업은 Dock이 여는 현재 작업 패널에서 진행합니다.

## 사용 흐름

1. 하단 Dock의 `📂 파일 열기`로 원본을 선택합니다.
2. 파일을 열면 자동 분석이 시작됩니다.
3. `✨ 추천`에서 후보를 생성합니다.
4. `🎯 후보`에서 마음에 드는 구간을 선택합니다.
5. `📱 미리보기`, `〰️ 파형`, `✂️ 컷`, `🎛️ 편집`, `📦 저장`으로 이어갑니다.

## 업데이트 확인

상단 버전 배지를 눌러 프로그램 정보 창을 열면 `🛰️ 업데이트 적용 상태`가 표시됩니다.

- `업데이트 확인`: 서비스워커 업데이트와 현재 DOM 버전을 다시 확인합니다.
- `이전 캐시 정리`: 이전 버전의 앱 셸 캐시를 정리합니다.
- `진단 복사`: 버전, 빌드 키, 캐시, 서비스워커, 엔진 프로필 정보를 JSON으로 복사합니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 모듈형 엔진

분석, 추천, 컷 감지, 렌더 품질, 세션 복구, 저장 큐를 독립 모듈로 분리해 유지보수와 확장을 쉽게 만들었습니다. v1.1.7에서는 버전/캐시 점검을 `Update Sentinel`이 담당하고, 브라우저 성능 프로필은 `Engine Boost Profile`이 담당합니다.
