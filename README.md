# AI 쇼츠 제작 스튜디오 v1.1.0

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠로 쓰기 좋은 후보 구간을 추천해 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.1.0 핵심

```text
파일 열기
→ 자동 분석
→ 추천 생성
→ 후보 비교/선택
→ 미리보기 후보 HUD 확인
→ 파형/컷/편집 조정
→ 저장 탭에서 품질 프리셋 선택
→ 쇼츠 저장
```

## 이번 버전 추가

- 후보 탭에 상위 후보 3개 비교 보드 추가.
- 후보 정렬 버튼 추가: 점수순 / 짧은 길이 / 빠른 시작.
- 후보 선택 후 미리보기 탭에 선택 후보 HUD 고정 표시.
- 선택 후보 HUD에서 후보 변경, 파형 조정, 저장 이동을 바로 실행.
- 기존 렌더 품질 프리셋, 저장 전 예상 시간/용량, 화면 떨림 방지 구조 유지.

## 실행

```bash
npm run serve
```

브라우저에서:

```text
http://localhost:8080
```

## 검수


결과:

```text

```

## 메모

이 프로젝트는 무료 로컬 브라우저 기능을 중심으로 동작합니다. 파일은 사용자의 브라우저 안에서 처리되며, 별도 서버 업로드를 전제로 하지 않습니다.

## 모듈형 엔진

분석, 추천, 컷 감지, 후보 비교, 미리보기 HUD, 렌더링, 저장 준비 안내, 렌더 품질 프리셋은 각각 독립 모듈로 분리되어 있습니다. 이후 무료 로컬 엔진을 더 강화하거나 고급 분석 모듈을 추가하기 쉽게 하기 위한 기반입니다.
