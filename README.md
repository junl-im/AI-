# AI 쇼츠 제작 스튜디오 v1.0.1

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠 후보 추천·후보 선택·세로 미리보기·편집·저장까지 이어주는 로컬 웹 스튜디오입니다.

## 이번 업데이트

- PC와 모바일 레이아웃을 분리했습니다.
- PC에서는 넓은 화면을 활용해 작업 폭을 확장하고, 하단 Dock은 8개 탭을 한 줄로 보여줍니다.
- 모바일에서는 기존처럼 4개씩 두 줄 탭 구조를 유지해 한손 조작을 우선합니다.
- 하단 Dock 첫 탭 이름을 `파일`에서 `파일 열기`로 변경했습니다.
- 상단 타이틀 영역에서는 프로그램명과 소개만 보여주고, 파일 열기/프로젝트 불러오기/자동 분석/편집 안내는 별도 시작 패널로 아래에 분리했습니다.
- 기존 시네마틱 히어로 배경, 필름 스트립, 카메라 프레임 연출은 유지했습니다.
- 새 반응형 CSS와 QA 가드를 추가했습니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 검수

```bash
npm run check
```

## 모듈형 엔진

분석, 추천, 컷, 렌더, 안정성 점검은 `src/engine/` 아래의 모듈형 엔진을 통해 연결됩니다. 새 기능은 가능한 한 앱 본문에 직접 붙이지 않고 엔진/렌더/UI 모듈로 분리하는 방향을 유지합니다.
