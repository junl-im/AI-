# AI 쇼츠 제작 스튜디오 v1.2.3

음악이나 영상을 브라우저 안에서 분석하고, 하이라이트 추천부터 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.2.3 핵심 패치

- 상단을 **쇼츠의 9:16 비율과 짧은 컷 타임라인**을 중심으로 다시 설계했습니다.
- 큰 제품명 아래에 `HOOK → BEAT → CAPTION → EXPORT` 흐름을 0:00~0:15 타임라인으로 표시합니다.
- PC 오른쪽에는 겹쳐진 세로 컷 프레임과 `새 쇼츠 만들기` 액션을 배치합니다.
- 모바일에서는 PC용 세로 프레임과 시작 액션을 숨기고 브랜드·설명·진행 순서만 간결하게 보여줍니다.
- 버전, 로컬 렌더 상태와 `DESIGNED BY 곰같은여우`를 한 줄의 작은 메타 정보로 통일했습니다.
- 기존 응답성 가드, 3열 Prime 작업실, 메뉴바 중심 모바일 흐름과 첫 화면 밖 페인트 지연은 유지합니다.

## 화면별 사용 흐름

### PC

상단의 `새 쇼츠 만들기` 또는 작업실의 `파일 열기`로 원본을 선택합니다. 첫 화면의 3열 Prime 작업실에서 추천 설정, 후보 목록, 세로 미리보기와 파형을 함께 확인하며 메뉴바는 해당 작업 카드로 빠르게 이동합니다.

### 모바일

상단에서 제품의 성격과 제작 순서를 확인한 뒤 하단 메뉴바의 `📂 파일 열기`를 눌러 시작합니다. 본문에는 중복 파일 불러오기 카드를 두지 않습니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다. 파일 탐색기에서 `index.html`을 직접 여는 것보다 로컬 서버 실행을 권장합니다.

## 검수

```bash
npm run check
```

v1.2.3 기준 문법, DOM 앵커, 분석·추천·렌더·저장 모듈, PC/모바일 레이아웃, 메뉴바 용어, 응답성 회귀 방지와 쇼츠 전용 상단을 포함해 105개 검사를 실행합니다.

## 업데이트 확인

상단 버전 텍스트를 누르면 현재 버전, 빌드 키, 서비스워커, 캐시와 엔진 프로필을 확인할 수 있습니다. 이전 캐시가 남아 있으면 정보 창에서 이전 캐시 정리를 실행한 뒤 새로고침합니다.

## 모듈형 엔진

분석, 추천, 컷 감지, 렌더 품질, 캐시와 안정성 감사 모듈의 계약은 유지합니다. 이번 패치는 상단 HTML·CSS, 메타 상태 동기화와 버전 동기화 중심이며 분석·렌더 엔진 로직은 변경하지 않았습니다.
