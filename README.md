# AI 쇼츠 제작 스튜디오 v1.0.6

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠 후보 추천·후보 선택·세로 미리보기·편집·저장까지 이어주는 로컬 웹 스튜디오입니다.

## 이번 업데이트

v1.0.6은 **Dock 탭 이동 시 화면 떨림을 잡는 모션 안정화 패치**입니다.

- Dock 탭을 눌렀을 때 여러 모듈이 동시에 스크롤을 실행하던 문제를 정리했습니다.
- 탭 이동 스크롤 담당을 `motion-stability.js` 한 곳으로 통합했습니다.
- `smooth scroll` 중복 실행을 없애고, 작업 패널을 안정적으로 한 번만 전면에 보이게 했습니다.
- 작업 패널 하이라이트는 유지하되, 화면이 흔들려 보이는 애니메이션은 줄였습니다.
- 추천/후보 안내 영역의 반짝임과 모션 충돌을 추가로 차단했습니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 검수

```bash
npm run check
```

현재 검수: 75/75 통과.

## 작업 흐름

```text
파일 열기
→ 자동 분석
→ 추천 생성
→ 후보 선택
→ 미리보기
→ 파형 / 컷 / 편집 / 저장
```

Dock은 작업 탭 내비게이션입니다. 각 탭을 누르면 해당 작업 패널이 화면 상단 근처로 안정적으로 이동합니다.

## 모듈형 엔진

분석, 추천, 컷, 렌더, 안정성 점검은 `src/engine/`, `src/render/`, `src/ui/` 아래 모듈로 분리합니다. 새 기능은 앱 본문에 직접 쌓지 말고 모듈과 QA를 함께 추가합니다.
