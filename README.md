## 목표

`AI 쇼츠 제작 스튜디오`는 사용자가 가진 풀 노래 또는 MP4/MOV/WebM 영상을 브라우저 안에서 분석하고, 쇼츠용 하이라이트 구간을 추천한 뒤 9:16 세로 영상으로 미리보기/내보내기하는 무료 웹 프로그램입니다.

v0.4.1은 사용자가 지적한 **파일 불러오기 후 UI가 틀어지는 문제**를 먼저 잡는 안정화 패치입니다. 상단 히어로 영역을 압축하고, 원본 audio/video 컨트롤이 레이아웃을 밀어내지 않도록 숨기며, 주요 작업을 하단 고정 Dock에 모았습니다.

서버 업로드 없이 로컬 브라우저에서 처리하는 것을 기본 원칙으로 합니다.

## 하단 Dock 설계 의도

하단 Dock은 단순 버튼 모음이 아니라, 파일을 불러온 뒤 실제 제작자가 가장 자주 누르는 흐름을 고정해둔 작업 레일입니다.

```text
파일 → 분석 → 추천 확인 → 편집 → 미리보기 → 썸네일 → 내보내기
```

상단에는 브랜드와 진행 단계만 남기고, 반복 작업 버튼은 하단에 고정했습니다. 모바일에서도 같은 Dock을 사용하므로 화면 폭에 따라 버튼이 가로 스크롤됩니다.

## 사용법

1. `npm run serve` 또는 정적 서버로 실행합니다.
2. 브라우저에서 `http://localhost:8080`을 엽니다.
3. 상단 또는 하단 Dock의 `파일` 버튼으로 오디오/비디오 파일을 엽니다.
4. 빠른 길이 칩이나 후보 길이 셀렉트를 선택합니다.
5. `분석하고 추천받기` 또는 하단 Dock의 `분석`을 누릅니다.
6. 추천 카드에서 구간을 선택합니다.
7. 하단 Dock의 `편집`을 누르면 구간/자막 편집 영역으로 이동합니다.
8. 파형 위 파란 선택 구간을 드래그하거나 시작/끝 핸들을 잡아 구간을 조절합니다.
9. 필요한 경우 시작/끝 숫자 입력, 1초 미세 조절 버튼, 현재 재생 위치 맞춤 버튼을 사용합니다.
10. 썸네일 템플릿과 자막을 적용합니다.
11. 하단 Dock의 `미리보기`, `썸네일`, `내보내기`를 사용합니다.
12. 작업 상태를 이어가려면 `프로젝트 저장`으로 JSON을 보관합니다.

## 프로젝트 JSON 주의사항

프로젝트 파일에는 원본 미디어가 포함되지 않습니다. 저장되는 항목은 다음과 같습니다.

- 추천 구간
- 직접 수정한 시작/끝 시간
- 자막 큐
- 자막 스타일/싱크 설정
- 썸네일 템플릿 설정
- 제목/해시태그
- 앱 설정

다시 열 때는 같은 원본 파일을 먼저 불러온 뒤 프로젝트 JSON을 적용하는 방식이 가장 안전합니다.

## 브라우저 주의사항

- MP4로 저장 가능 여부는 브라우저의 `MediaRecorder` 지원 범위에 따라 달라집니다.
- Chrome 계열은 WebM 내보내기가 안정적입니다.
- Safari/iOS는 `captureStream`, `MediaRecorder`, 비디오 코덱 지원이 제한될 수 있습니다.
- 여러 후보를 일괄 저장할 때 브라우저가 여러 파일 다운로드 허용을 물어볼 수 있습니다.
- 이 앱은 서버에 파일을 업로드하지 않습니다.
