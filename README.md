# AI 쇼츠 제작 스튜디오 v1.1.9

음악이나 영상을 브라우저 안에서 분석하고, 쇼츠 후보 추천부터 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.1.9 핵심 패치

- PC를 탭 하나씩 보는 화면에서 **3열 Prime 작업실**로 전환했습니다.
- PC 첫 작업 구간에서 불러오기·추천·미리보기·후보·파형을 동시에 확인할 수 있습니다.
- 컷·정밀 편집·저장·프로젝트 영역은 같은 분할 그리드 아래로 자연스럽게 이어집니다.
- 모바일 첫 화면에서 중복 파일 열기/프로젝트 불러오기 카드를 제거했습니다.
- 모바일은 하단 Dock의 파일 열기를 사용하고, 본문에는 4단계 진행 순서만 표시합니다.
- 상단 버전은 왼쪽 끝, `Design by 곰같은여우`는 같은 줄 오른쪽 끝에 고정했습니다.
- 제품 설명은 최대 폭을 제한하고 좌우 안쪽 여백을 늘렸습니다.
- v1.1.8의 이벤트 기반 UI 동기화와 저사양 성능 가드는 그대로 유지합니다.

## 화면별 사용 흐름

### PC

첫 화면의 분할 작업실에서 원본을 열고 추천 설정, 후보 목록, 세로 미리보기, 파형을 함께 확인합니다. 하단 Dock은 탭 전환용이라기보다 해당 작업 카드로 빠르게 이동하는 내비게이션으로 동작합니다.

### 모바일

첫 화면에서는 제작 순서만 확인합니다. 하단 Dock의 `📂 파일 열기`를 누르면 파일 선택기가 바로 열리고, 이후 현재 단계의 패널만 표시됩니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 검수

```bash
npm run check
```

v1.1.9 기준 문법, DOM 앵커, 렌더/추천/저장 모듈, PC 분할 레이아웃, 모바일 진행 화면, 성능 가드까지 102개 검사를 실행합니다.

## 업데이트 확인

상단 버전 배지를 누르면 현재 버전, 빌드 키, 서비스워커, 캐시, 엔진 프로필을 확인할 수 있습니다. 이전 캐시 정리와 진단 복사 기능도 유지됩니다.

## 모듈형 엔진

분석, 추천, 컷 감지, 렌더 품질, 캐시, 안정성 감사 모듈의 계약은 변경하지 않았습니다. v1.1.9은 화면 구조만 기기별로 분리해 기존 제작 기능을 그대로 사용합니다.
