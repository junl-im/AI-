# AI 쇼츠 제작 스튜디오 v1.6.5

브라우저에서 미디어 분석, 쇼츠 후보 추천, 자막 편집, 세로 렌더링과 선택형 Local AI를 수행하는 로컬 우선 제작 도구입니다. v1.6.5는 피사체 움직임을 따라가는 자막 안전 스마트 리프레임을 추가합니다.

## 핵심 기능

- 모듈형 엔진 기반 오디오·영상 분석과 쇼츠 후보 추천
- 자동 컷, 후보 고정, 자막·SRT 편집, 9:16 렌더 큐
- 스마트 피사체 추적과 자막 안전 동적 크롭
- 옵션 signature 기반 분석 캐시와 namespace 선택 정리
- 세션 순환 백업과 오프라인 앱 셸 무결성 복구
- Ollama·llama.cpp 구조화 카피 생성
- whisper.cpp·OpenAI 호환 로컬 전사 및 SRT 생성

## 스마트 리프레임 사용

1. 영상을 불러오고 자동 분석을 완료합니다.
2. 세로 프레임에서 `스마트 피사체 추적`을 선택합니다.
3. 기본적으로 영상 움직임 중심을 따라가는 추적선이 즉시 생성됩니다.
4. 브라우저 얼굴 감지 또는 MediaPipe detector가 준비된 환경에서는 `얼굴 추적 시도`로 얼굴 중심 추적을 적용합니다.
5. `자막 안전 영역 피하기`를 켜면 자막 위치에 따라 피사체의 세로 배치를 조정합니다.

얼굴 감지를 사용할 수 없어도 모션 추적 또는 중앙 크롭으로 안전하게 대체되며 기존 편집·렌더 기능은 중단되지 않습니다.

## 로컬 우선 원칙

- 외부 CDN이나 원격 비전 API를 자동 호출하지 않습니다.
- AI 서버 요청은 사용자의 명시적 동작 뒤에만 실행됩니다.
- Local AI와 얼굴 감지 모듈이 없어도 기본 기능은 계속 동작합니다.
- 분석 프레임과 얼굴 이미지는 프로젝트에 저장하지 않습니다.
- 선택형 모듈은 필요한 작업 단계에서만 지연 적재합니다.

## 실행과 검수

```bash
npm run serve
npm test
```

브라우저에서 `http://127.0.0.1:8080`을 엽니다. 등록 자동 검사 228개와 스마트 리프레임 전용 회귀를 포함합니다.

## 패키징

```bash
npm run package:full
PATCH_BASE_REF=823d608 PATCH_FROM_VERSION=1.6.4 npm run package:patch
```

전체 ZIP은 실행·문서·QA 파일을 포함하고 `.git`, `node_modules`, 중첩 `dist`를 제외합니다. 패치 ZIP은 v1.6.4 기준 변경·신규 파일만 포함합니다.

## 알려진 제한

- MediaPipe 모델과 WASM 파일은 배포본에 포함되지 않습니다.
- 얼굴 감지 미지원 환경에서는 모션 중심 추적을 사용합니다.
- 다중 화자 음성 분리와 얼굴 연결은 아직 제공하지 않습니다.
- 실기기 Safari·Samsung Internet, 물리 GPU·NPU는 별도 검증이 필요합니다.
