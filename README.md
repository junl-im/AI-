# AI 쇼츠 제작 스튜디오 v1.2.8

음악이나 영상을 브라우저 안에서 분석하고, 하이라이트 추천부터 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.2.8 핵심 패치

- 상단 중앙의 `LOCAL · PRIVATE · 9:16` 상태 문구와 점 표시를 제거했습니다.
- 상단 메타를 왼쪽 `BUILD v1.2.8 · 모바일·PC 호환`, 오른쪽 `DESIGNED BY 곰같은여우`의 두 축으로 단순화했습니다.
- 실행 중 중앙 상태 영역을 다시 생성하던 `FlowCommandBridge` 로직도 함께 제거했습니다.
- 720px 이하와 390px 이하에서도 `BUILD`와 `DESIGNED BY` 라벨이 숨지 않도록 모바일 전용 메타 레일을 추가했습니다.
- 기존 전용 SVG 아이콘, PC 3열 작업실, 모바일 진행 중심 화면과 단계형 로딩 구조는 유지합니다.
- 버전·빌드 키·서비스워커 캐시를 v1.2.8 / `1.2.8-header-meta`로 동기화했습니다.
- 전체 설치용 ZIP과 v1.2.7 덮어쓰기용 패치 ZIP을 함께 제공합니다.

## 배포 파일

```bash
npm run package
```

- `dist/ai-shorts-studio-v1.2.8-release.zip`: 전체 설치용
- `dist/ai-shorts-studio-v1.2.8-patch-from-v1.2.7.zip`: v1.2.7 덮어쓰기용

패치 ZIP은 기존 v1.2.7 폴더 최상위에 풀고 파일 교체를 허용합니다. 다른 버전에 적용하지 않습니다.

## 화면별 사용 흐름

### PC

상단 왼쪽에서 빌드와 기기 호환 정보를 확인하고, 오른쪽에서 디자인 서명을 확인할 수 있습니다. `새 쇼츠 만들기` 또는 작업실의 `파일 열기`로 시작하며 진행 단계가 바뀌면 메뉴바 활성 상태와 현재 작업 카드가 함께 이동합니다.

### 모바일

작은 화면에서도 `BUILD v1.2.8`, `모바일 · PC 호환`, `DESIGNED BY 곰같은여우`가 같은 상단 줄에 유지됩니다. 파일 선택은 하단 메뉴바의 `파일 열기`가 담당하고 본문에는 중복 실행 버튼 대신 제작 순서만 표시합니다.

## 아이콘 자산 구조

- SVG 원본: `assets/icons/studio/`
- CSS 매핑: `assets/css/icon-system.css`
- 사용 방식: `data-icon="upload"` 또는 `<span class="studio-icon" data-icon="upload"></span>`

핵심 UI에는 운영체제별 이모지나 문자 기호 대신 24×24 전용 SVG 자산을 사용합니다.

## 모듈형 엔진

오디오 특징 추출, 영상 모션 분석, 자동 컷, 추천 점수, 렌더 품질과 안정성 감사를 독립 모듈로 유지합니다. v1.2.8은 분석·추천·렌더 엔진 계약을 변경하지 않고 상단 메타 표현만 개선했습니다.

## 단계별 UI 로딩

| 단계 | 준비되는 기능 | 시점 |
|---|---|---|
| Core | 앱 상태, 분석·추천·렌더 핵심, 메뉴 내비게이션 | 첫 화면 |
| Shell | 흐름 보정, UX 동기화, 작업 공간 안정화, 세션 연속성 | 유휴 시간 또는 첫 접근 |
| Editing | 범위 조절, 후보 비교·고정, 저장 준비, 품질 계획 | 후보·미리보기·편집 접근 |
| Export | 저장 완료 센터 | 내보내기 접근 |
