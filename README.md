## 목표

`AI 쇼츠 제작 스튜디오`는 사용자가 가진 풀 노래 또는 MP4/MOV/WebM 영상을 브라우저 안에서 분석하고, 쇼츠용 하이라이트 구간을 추천한 뒤 9:16 세로 영상으로 미리보기/내보내기하는 무료 웹 프로그램입니다.

v0.3.0은 기능을 무리하게 늘리기보다 실제 사용 흐름을 편하게 만드는 패치입니다. 첫 화면에서 파일 열기/바로 분석/프로젝트 불러오기를 바로 누를 수 있고, 제작 단계 표시, 빠른 길이 선택, 상단 작업 도크, 모바일 하단 액션바, 1초 구간 미세 조절을 추가했습니다.

서버 업로드 없이 로컬 브라우저에서 처리하는 것을 기본 원칙으로 합니다.

## v0.3.0 핵심 개선

- 히어로 영역 빠른 시작 버튼: 파일 열기, 바로 분석, 프로젝트 불러오기
- 4단계 제작 진행 표시: 불러오기 → AI 추천 → 편집 → 내보내기
- 후보 길이 빠른 선택 칩: 자동, 15초, 30초, 60초, 90초
- 미리보기/내보내기 상단 액션 도크
- 모바일 하단 고정 액션바
- 시작/끝 시간 1초 미세 조절 버튼
- 추천 카드 hover/selected 상태 개선
- 포커스 링과 터치 타깃 개선
- 기존 자막, 썸네일, 프로젝트 저장 기능 유지

## 무료 기능 기준의 현실적 범위

이 버전은 유료 생성형 AI API를 사용하지 않습니다. 추천은 다음 로컬 분석값으로 계산합니다.

- RMS 에너지
- 피크 밀도
- 트랜지언트 변화량
- 무음/저에너지 비율
- 에너지 상승 구간
- 비디오 프레임 차이 기반 움직임 점수

따라서 “AI가 새 영상을 생성”하는 방식이 아니라, **사용자 파일에서 쇼츠로 쓸 만한 구간을 자동 큐레이션하고 세로 영상으로 편집하는 방식**입니다.

## 사용법

1. `npm run serve` 또는 정적 서버로 실행합니다.
2. 브라우저에서 `http://localhost:8080`을 엽니다.
3. 첫 화면의 `파일 열기` 또는 불러오기 영역에서 오디오/비디오 파일을 엽니다.
4. 빠른 길이 칩이나 후보 길이 셀렉트를 선택합니다.
5. `분석하고 추천받기` 또는 상단 `바로 분석`을 누릅니다.
6. 추천 카드에서 구간을 선택합니다.
7. 필요한 경우 시작/끝 시간을 수정하거나 1초 미세 조절 버튼을 사용한 뒤 `구간 적용`을 누릅니다.
8. SRT/VTT 자막을 열거나 자막 텍스트를 붙여넣고 `자막 적용`을 누릅니다.
9. 상단 작업 도크, 미리보기 카드, 모바일 하단 액션바 중 편한 곳에서 미리보기/내보내기를 실행합니다.
10. 작업 상태를 이어가려면 `프로젝트 저장`으로 JSON을 보관합니다.

## 프로젝트 JSON 주의사항

프로젝트 파일에는 원본 미디어가 포함되지 않습니다. 저장되는 항목은 다음과 같습니다.

- 추천 구간
- 직접 수정한 시작/끝 시간
- 자막 큐
- 자막 스타일/싱크 설정
- 제목/해시태그
- 앱 설정

다시 열 때는 같은 원본 파일을 먼저 불러온 뒤 프로젝트 JSON을 적용하는 방식이 가장 안전합니다.

## 브라우저 주의사항

- MP4로 저장 가능 여부는 브라우저의 `MediaRecorder` 지원 범위에 따라 달라집니다.
- Chrome 계열은 WebM 내보내기가 안정적입니다.
- Safari/iOS는 `captureStream`, `MediaRecorder`, 비디오 코덱 지원이 제한될 수 있습니다.
- 이 앱은 서버에 파일을 업로드하지 않습니다.

## 저작권 안내

업로드한 음원과 영상의 사용 권리는 사용자가 확인해야 합니다. 이 프로그램은 편집과 추천을 돕는 도구이며, 플랫폼 게시 권리나 Content ID 문제를 자동으로 해결하지 않습니다.
