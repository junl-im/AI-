# AI 쇼츠 제작 스튜디오 v1.2.1

음악이나 영상을 브라우저 안에서 분석하고, 쇼츠 후보 추천부터 세로 미리보기와 저장까지 이어주는 로컬 웹 스튜디오입니다.

## v1.2.1 핵심 패치

- 첫인상을 책임지는 상단을 **시네마틱 스튜디오 컨트롤 데크**로 전면 개편했습니다.
- PC에서는 왼쪽의 제품 소개와 오른쪽의 `STUDIO READY` 시작 패널을 동시에 보여줍니다.
- 상단 `새 프로젝트 시작`은 기존 파일 입력을 그대로 사용하므로 작업 흐름과 상태가 이중화되지 않습니다.
- 로컬 분석, 하이라이트 자동 추천, 9:16 렌더를 짧은 기능 칩으로 보여줍니다.
- 모바일에서는 시작 패널을 숨기고 브랜드 소개만 압축해, 파일 선택은 기존 하단 메뉴바가 전담합니다.
- 저사양 모드와 모션 감소 환경에서는 회전·색상 흐름·블러 효과를 자동으로 줄입니다.
- v1.2.0에서 해결한 Observer 피드백 루프와 페이지 응답성 가드는 그대로 유지합니다.

## 화면별 사용 흐름

### PC

상단의 `새 프로젝트 시작` 또는 작업실의 `파일 열기`로 원본을 선택합니다. 첫 화면의 3열 Prime 작업실에서 추천 설정, 후보 목록, 세로 미리보기와 파형을 함께 확인하며, 하단 메뉴바는 해당 작업 카드로 빠르게 이동합니다.

### 모바일

상단은 브랜드와 제품 설명만 간결하게 표시합니다. 첫 화면에서 4단계 제작 순서를 확인하고, 하단 메뉴바의 `📂 파일 열기`를 눌러 작업을 시작합니다.

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다. 파일 탐색기에서 `index.html`을 직접 여는 것보다 로컬 서버 실행을 권장합니다.

## 검수

```bash
npm run check
```

v1.2.1 기준 문법, DOM 앵커, 렌더·추천·저장 모듈, PC/모바일 레이아웃, 런타임 피드백 루프 방지와 새 상단 컨트롤 데크까지 104개 검사를 실행합니다.

## 업데이트 확인

상단 버전 배지를 누르면 현재 버전, 빌드 키, 서비스워커, 캐시와 엔진 프로필을 확인할 수 있습니다. 이전 캐시가 남아 있으면 정보 창에서 이전 캐시 정리를 실행한 뒤 새로고침합니다.

## 모듈형 엔진

분석, 추천, 컷 감지, 렌더 품질, 캐시와 안정성 감사 모듈의 계약은 유지합니다. 이번 패치는 상단 HTML·CSS와 버전 동기화 중심이며, 분석·렌더 엔진 로직은 변경하지 않았습니다.
