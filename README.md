# AI 쇼츠 제작 스튜디오 v1.0.0

음악이나 영상을 열면 브라우저 안에서 자동 분석하고, 쇼츠 후보 추천·후보 선택·세로 미리보기·편집·저장까지 이어주는 로컬 웹 스튜디오입니다.

## 이번 업데이트

- 상단 프로그램명 영역에 쇼츠 전용 시네마틱 히어로 디자인 적용
- 기존 샘플의 움직이는 배경 감성을 참고하되, 마스터링 물결 대신 필름 스트립·카메라 프레임·빛 스윕 연출로 재해석
- 상단 좌측 `v1.0.0`, 우측 `Design by 곰같은여우` 한 줄 유지
- 배경은 기존 어두운 글래스 느낌을 유지하면서 영상/쇼츠 분위기에 맞게 강화
- `prefers-reduced-motion` 환경에서는 애니메이션을 자동 완화
- 기존 초연결 흐름과 하단 Dock 구조 유지

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 검수

```bash
npm run check
```

## 모듈형 엔진

분석, 추천, 컷, 렌더, 안정성 점검은 `src/engine/` 아래의 모듈형 엔진을 통해 연결됩니다. 새 기능은 가능한 한 앱 본문에 직접 붙이지 않고 엔진/렌더/UI 모듈로 분리하는 방향을 유지합니다.
