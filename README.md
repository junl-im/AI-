# AI 쇼츠 제작 스튜디오 v0.8.1 - Lean Dock UI/UX 성능 패치

무료 브라우저 기능 기반으로 노래/영상에서 쇼츠 후보를 추천하고 9:16 세로 결과물을 내보내는 웹 스튜디오입니다. v0.8.1은 하단 Dock을 핵심 2버튼으로 단순화하고, 파일 로드 후 UI 안정성과 반응성을 개선한 패치입니다.

## v0.8.1 핵심

- 하단 Dock 버튼을 7개에서 2개로 축소
- 📂 파일 열기 / ⚡ 분석하기 버튼을 50:50 반반 배치
- Dock 버튼 크기 확대 및 모바일 터치 영역 개선
- Dock 반복 동기화 타이머 제거
- Dock 상태 동기화를 `requestAnimationFrame` 기반으로 변경
- 미리보기 정지 화면 렌더링을 프레임 단위로 묶어 중복 호출 감소
- 무거운 카드/리스트 영역에 CSS containment 적용
- 파일 로드 후 원본 미디어 컨트롤 숨김 안정화 유지

## 기존 유지 기능

- 오디오/영상 로컬 분석
- AI 추천 구간 카드
- 파형 드래그 핸들
- 파형 위 컷 마커
- 자동 컷 보정
- 자막 SRT/VTT 및 자막 디자인 패널
- 결과물 품질 패널
- 썸네일 템플릿과 PNG 저장
- 단일/일괄 내보내기

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080` 을 엽니다.

## 검수

```bash
npm run check
```

## 주의

내보내기는 브라우저 MediaRecorder 지원 범위에 따라 WebM 또는 MP4 계열로 저장됩니다. iOS Safari, 인앱 브라우저, 저사양 모바일에서는 일부 품질 옵션이나 다운로드 방식이 제한될 수 있습니다.

## 이전 v0.8.0 유지 기능

v0.8.0에서 추가된 파형 컷 마커 편집 기능은 유지됩니다. 비트, 장면 전환, 무음 종료 지점을 색상 마커로 구분하고, 마커를 클릭하면 해당 시간으로 이동합니다. 선택 구간 밖의 마커를 클릭하면 가까운 시작점 또는 끝점을 컷 위치에 맞춰서 쇼츠 구간을 빠르게 보정할 수 있습니다.
