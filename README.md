# AI 쇼츠 제작 스튜디오 v0.7.0

무료 브라우저 기능 기반으로 노래/영상에서 쇼츠 후보를 추천하고 9:16 세로 결과물을 내보내는 웹 스튜디오입니다. v0.7.0은 결과물 품질 패치입니다.

## v0.7.0 핵심

- 밝기/대비/채도 보정
- 비네팅 강도 조절
- 오디오 페이드 인/아웃
- 인트로/아웃트로 텍스트 오버레이
- 워터마크/로고 텍스트와 위치 설정
- 미리보기 안전영역 가이드 토글
- 제목/해시태그 다시 추천 버튼
- 품질 설정을 프로젝트 JSON에 저장/불러오기

## 기존 유지 기능

- 오디오/영상 로컬 분석
- AI 추천 구간 카드
- 파형 드래그 핸들
- 자막 SRT/VTT 및 자막 디자인 패널
- 썸네일 템플릿과 PNG 저장
- 단일/일괄 내보내기
- 하단 고정 Dock

## 실행

```bash
npm run serve
```

브라우저에서 `http://localhost:8080` 을 엽니다.

## 검수

```bash
npm run check
```

## 주의

내보내기는 브라우저 MediaRecorder 지원 범위에 따라 WebM 또는 MP4 계열로 저장됩니다. iOS Safari, 인앱 브라우저, 저사양 모바일에서는 일부 품질 옵션이나 다운로드 방식이 제한될 수 있습니다.


## v0.7.0 자동 컷 편집 패치

이번 버전은 무음, 비트 트랜지언트, 영상 장면 전환을 함께 계산해서 컷 후보 타임라인을 표시합니다. 추천 카드에는 템포 점수와 무음 위험이 반영되고, 선택 구간 또는 전체 후보를 자동 컷 기준으로 보정할 수 있습니다.

- 무음 구간 자동 감지
- 비트 컷 포인트 추천
- 영상 장면 전환 컷 후보 표시
- 선택 구간 자동 보정
- 모든 추천 후보 일괄 컷 보정
- 자동 컷 옵션 프로젝트 설정 저장
