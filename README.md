# AI 쇼츠 제작 스튜디오 v0.9.0

Design by **곰같은여우**.

무료 로컬 브라우저 기능만으로 오디오/영상 파일을 분석하고, 쇼츠 후보 구간을 추천하고, 세로 미리보기/자막/썸네일/내보내기를 수행하는 웹 스튜디오입니다.

## v0.9.0 핵심 변경 - 모듈형 엔진 강화

이번 버전의 핵심은 기능을 더 붙이는 것보다 **엔진을 분리하고 강화하는 것**입니다. 앞으로 기능이 커져도 무너지지 않게 분석, 추천, 컷, 렌더, UI 피드백을 독립 모듈로 나눴습니다.

### 추가된 엔진 구조

```text
src/engine/
├─ module-registry.js       # 엔진 모듈 등록/스냅샷/훅 실행
├─ performance-budget.js    # 파일 크기/길이/기기 성능 기반 자동 성능 예산
├─ analysis-pipeline.js     # 오디오/영상/자동 컷 분석 파이프라인
├─ scoring-pipeline.js      # 추천 점수 보강, 품질 게이트, 엔진 배지
└─ engine-kernel.js         # 앱에서 호출하는 엔진 통합 진입점
```

### 강화된 부분

- 오디오 분석, 영상 움직임 분석, 자동 컷 감지, 추천 점수 계산을 독립 모듈로 분리
- 파일 크기와 기기 성능에 따라 `안전 모드 / 균형 성능 / 최대 성능` 예산 자동 선택
- 추천 후보에 품질 게이트 적용
- 무음 위험, 비트감, 상승감, 움직임 기반으로 추천 점수 재정렬
- 추천 카드에 엔진 배지 표시
- 엔진 상태 카드 추가
- 서비스워커 캐시와 QA에 엔진 모듈 추가

## 실행

```bash
npm run serve
```

브라우저에서 열기:

```text
http://localhost:8080
```

## 검수

```bash
npm run check
```

현재 QA는 모듈형 엔진 구조, 기존 UI/UX, 자막, 컷 마커, Dock, 렌더링 앵커를 함께 검사합니다.

## 설계 원칙

```text
무료 기능 기반
서버 업로드 없음
브라우저 로컬 분석
모듈형 엔진
기능 추가보다 안정성 우선
모바일 UI 붕괴 방지
하단 Dock은 📂 파일 열기 / ⚡ 분석하기만 유지
```
