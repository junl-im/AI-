# AI 쇼츠 제작 스튜디오 v0.8.2

**Design by 곰같은여우** 브랜드/피드백 UX 패치입니다.  
무료 로컬 기능 기반으로 노래 또는 MP4/WebM/MOV 파일을 분석해 쇼츠 후보를 추천하고, 세로 미리보기/자막/썸네일/내보내기까지 수행하는 웹 프로그램입니다.

## v0.8.2 핵심 변경

- 상단 브랜딩에 `Design by 곰같은여우` 시그니처 추가
- 하단 Dock은 요청대로 `📂 파일 열기` / `⚡ 분석하기` 두 개 버튼만 유지
- 두 Dock 버튼은 50:50 반반 넓은 터치 영역 유지
- 버튼/라벨/추천 카드 클릭 시 가벼운 리플과 눌림 피드백 추가
- `navigator.vibrate` 기반 햅틱 진동 알림 추가
- 파일 열기, 분석, 저장/내보내기, 복사, 성공, 경고, 오류별 진동 패턴 구분
- 토스트 알림을 성공/경고/오류/파일/내보내기/복사 유형별 아이콘과 색상으로 구분
- 기존 v0.8.1 성능 최적화 유지: RAF 동기화, Dock 비폴링 구조, CSS containment 유지

## 실행

```bash
npm run serve
```

브라우저에서 아래 주소를 엽니다.

```text
http://localhost:8080
```

## 검수

```bash
npm run check
```

## 햅틱 참고

진동 알림은 브라우저와 기기에서 `navigator.vibrate`를 지원할 때만 동작합니다.  
iOS Safari, 일부 인앱 브라우저, 데스크톱 브라우저에서는 진동이 무시될 수 있으며, 이 경우에도 토스트 색상/아이콘 피드백은 유지됩니다.

## 기존 기능 유지

- 로컬 오디오/영상 분석
- 쇼츠 추천 카드
- 파형 컷 마커
- 무음/비트/장면 전환 컷 포인트
- 추천 구간 드래그 핸들
- 자막 SRT/VTT 및 스타일 패널
- 품질 보정 패널
- 썸네일 템플릿
- 단일/일괄 내보내기
- 프로젝트 JSON 저장/불러오기
