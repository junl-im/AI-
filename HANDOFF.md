# HANDOFF - AI Shorts Studio v0.4.0

## 현재 상태

- 정적 웹앱으로 실행 가능
- 외부 CDN 의존성 없음
- 오디오/비디오 파일 로컬 분석 가능
- 추천 카드 및 파형 표시 가능
- 추천 구간 시작/끝 수동 조절 가능
- 1초 단위 시작/끝 미세 조절 가능
- 파형 위 드래그 핸들로 구간 조절 가능
- 현재 재생 위치로 시작점 맞춤 가능
- SRT/VTT 자막 업로드/붙여넣기 가능
- 자막 스타일 및 싱크 보정 가능
- 썸네일 템플릿 4종 선택 가능
- 9:16 canvas preview 가능
- 썸네일 PNG 저장 가능
- 추천 후보 일괄 내보내기 가능
- 프로젝트 JSON 저장/불러오기 가능
- MediaRecorder 기반 export 가능
- 히어로 빠른 시작 버튼 유지
- 4단계 제작 진행 표시 유지
- 빠른 길이 선택 칩 유지
- 상단 작업 도크 유지
- 모바일 하단 액션바 유지
- QA smoke test 29/29 통과

## 시작 방법

```bash
npm run serve
```

브라우저에서 `http://localhost:8080` 접속.

## 검수 순서

1. 첫 화면의 `파일 열기` 버튼과 불러오기 카드의 `파일열기`가 모두 동작하는지 확인
2. MP3, WAV 또는 MP4 파일 업로드
3. 제작 단계 표시가 `AI 추천` 단계로 넘어가는지 확인
4. 빠른 길이 칩을 눌렀을 때 후보 길이 셀렉트가 바뀌는지 확인
5. `바로 분석` 또는 `분석하고 추천받기` 클릭
6. 추천 카드 3개 이상 표시 확인
7. 카드 클릭 시 파형 하이라이트와 제목/해시태그 갱신 확인
8. 파형 위 파란 선택 구간과 양쪽 핸들이 표시되는지 확인
9. 선택 구간을 드래그해서 전체 위치가 이동하는지 확인
10. 왼쪽/오른쪽 핸들로 시작/끝 시간이 바뀌고 자동 적용되는지 확인
11. `현재 재생 위치로 시작 맞춤` 버튼이 선택 구간 시작점을 이동시키는지 확인
12. 시작/끝 시간을 수정하고 `구간 적용` 클릭
13. 1초 미세 조절 버튼이 시작/끝 시간을 바꾸고 구간에 반영되는지 확인
14. 썸네일 템플릿 4종을 바꿀 때 미리보기 디자인이 바뀌는지 확인
15. SRT/VTT 파일 업로드 또는 자막 텍스트 붙여넣기 후 `자막 적용` 클릭
16. 자막 상태가 `N개 자막`으로 바뀌는지 확인
17. `미리보기` 버튼이 원본 버튼, 상단 도크, 모바일 액션바에서 같은 동작을 하는지 확인
18. `썸네일 저장`으로 선택 템플릿이 반영된 PNG 다운로드 확인
19. `후보 일괄 내보내기`로 상위 3개 또는 5개가 순차 저장되는지 확인
20. `프로젝트 저장`으로 JSON 다운로드 확인
21. `프로젝트 불러오기`로 추천/자막/설정/썸네일 템플릿 복원 확인
22. `내보내기` 버튼이 원본 버튼, 상단 도크, 모바일 액션바에서 같은 동작을 하는지 확인
23. `진단 복사` 버튼으로 JSON 복사 확인

## 알려진 제한

- 브라우저에 따라 MP4 대신 WebM으로 저장된다.
- iOS Safari에서는 MediaRecorder와 captureStream 지원이 제한될 수 있다.
- 매우 긴 4K 영상은 브라우저 메모리 한계로 분석/내보내기가 느릴 수 있다.
- 비디오의 오디오 트랙 디코딩은 브라우저 코덱 지원에 따라 실패할 수 있다. 실패 시 비디오 움직임 분석 중심으로 추천한다.
- 프로젝트 JSON에는 원본 미디어가 포함되지 않는다.
- 자막 자동 생성은 음성인식이 아니라 텍스트 분할 fallback이다.
- 여러 후보 일괄 내보내기는 브라우저의 다중 다운로드 허용 설정에 영향을 받는다.
- 모바일 하단 액션바는 화면 폭이 좁을 때만 표시된다.

## 배포 메모

- GitHub Pages에 그대로 올릴 수 있다.
- Firebase Hosting에도 정적 파일로 배포 가능하다.
- `.nojekyll` 포함.
- Service Worker 캐시 버전은 `sw.js`, `index.html`, README의 cache key를 같이 갱신한다.
- 새 스크립트를 추가하면 `sw.js`의 `SHELL_FILES`에도 추가해야 한다.
- v0.4.0에서 추가된 `assets/css/advanced-editor.css`, `src/ui/range-drag-controls.js`는 Service Worker 캐시에 포함되어 있다.
