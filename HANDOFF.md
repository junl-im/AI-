# HANDOFF - AI 쇼츠 제작 스튜디오 v1.1.0

## 이번 버전 목적

v1.1.0은 후보 선택과 미리보기 단계를 더 상용 프로그램처럼 만드는 버전입니다. 사용자가 추천 생성 후 후보를 더 쉽게 비교하고, 선택한 후보가 미리보기/파형/저장으로 확실하게 이어지도록 했습니다.

## 핵심 변경

```text
추천 생성
→ 후보 탭 Candidate Pro Board 표시
→ 상위 후보 비교 / 정렬
→ 후보 선택
→ 미리보기 HUD에 선택 정보 고정
→ 파형 조정 / 저장으로 연결
```

## 확인해야 할 흐름

1. 파일 열기 후 자동 분석된다.
2. 추천 탭에서 추천 생성을 누른다.
3. 후보 탭에 Candidate Pro Board가 표시된다.
4. 점수순 / 짧은 길이 / 빠른 시작 정렬 버튼이 보인다.
5. 상위 3개 비교 카드 또는 기존 후보 카드를 클릭하면 선택된다.
6. 후보 선택 후 미리보기 탭으로 자연스럽게 연결된다.
7. 미리보기 탭에 선택 후보 HUD가 표시된다.
8. HUD에서 후보 변경 / 파형 조정 / 저장 이동이 가능하다.
9. Dock 클릭 시 화면 떨림 없이 해당 구간으로 이동한다.

## 주의

후보 비교 보드는 기존 추천 카드의 선택 로직을 그대로 호출합니다. 따라서 후보 선택의 단일 진입점은 유지되고, 미리보기/저장/파형 연결도 기존 초연결 흐름과 충돌하지 않도록 설계되어 있습니다.

## 알려진 제한

```text
├─ 후보 비교 보드는 브라우저 DOM 상태를 기준으로 동기화되므로 아주 긴 후보 목록에서는 갱신이 지연될 수 있음
├─ iOS Safari/인앱 브라우저에서는 일부 다운로드 또는 진동 피드백이 제한될 수 있음
├─ 긴 영상은 기기 메모리와 브라우저 성능에 영향을 받음
├─ MediaRecorder의 MP4/WebM 지원은 브라우저별로 다름
└─ 자동 추천은 로컬 분석 기반이며 저작권/게시 권한은 사용자가 확인해야 함
```
