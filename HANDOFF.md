# HANDOFF - AI 쇼츠 제작 스튜디오 v0.9.3

## 이번 버전 핵심

v0.9.3은 안정화 및 엔진 구조 보강 패치입니다. 새 UI를 대량 추가하기보다, 이후 상용급 확장을 견딜 수 있도록 엔진 내부를 더 모듈화했습니다.

## 유지해야 할 UX 원칙

- 상단은 프로그램 소개만 보여줍니다. 패치 노트나 내부 엔진 용어를 올리지 않습니다.
- 파일을 열면 자동 분석합니다. 별도 분석 버튼을 사용자에게 노출하지 않습니다.
- 사용자는 `✨ 추천 생성` 버튼으로 후보 생성을 시작합니다.
- 하단 Dock은 8개 탭 퀵 메뉴입니다. 각 기능 화면은 한 탭씩 보여줍니다.
- 후보 선택 후 미리보기 탭으로 연결되는 초연결 흐름을 유지합니다.

## 새 엔진 모듈

- `module-contracts.js`: 모듈 등록값과 분석/추천 결과의 최소 계약을 검증합니다.
- `analysis-cache.js`: 같은 파일을 반복 처리할 때 브라우저 세션 캐시를 사용합니다.
- `pro-engine-tuner.js`: 추천 후보 신뢰도, 등급, 성능 예산 힌트를 계산합니다.
- `stability-auditor.js`: 앱 상태와 엔진 상태의 불일치를 점검합니다.

## 주의

캐시는 메모리 기반입니다. 새로고침하면 사라지며, 원본 파일을 저장하지 않습니다.

## 검수

`npm run check` 기준 전체 QA를 통과해야 릴리스합니다.


## 검수 순서

1. `npm run check`를 실행합니다.
2. 파일 열기 즉시 자동 분석되는지 확인합니다.
3. 추천 생성 후 후보 선택 시 미리보기 탭으로 전환되는지 확인합니다.
4. 하단 Dock 8탭 이동과 저장 탭 내보내기를 확인합니다.

## 알려진 제한

- 브라우저/기기 성능에 따라 분석 속도가 달라질 수 있습니다.
- 일부 iOS/인앱 브라우저는 진동, MediaRecorder, 다운로드 동작을 제한할 수 있습니다.
- 세션 분석 캐시는 새로고침하면 사라지며 원본 파일을 저장하지 않습니다.
